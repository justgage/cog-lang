Definitions

Standard out

Usually the main output of a console. Some place specified for most output of the program to go.

The Primitives

The basic built in types of the langauge including: Floats, Strings, Lists and Booleans

Requirements

id functional-requirement demonstration scenarios success measure
0The programing language SHALL be able to print the primitives to standard out
  1. valid code was ran through the executer that contained a print statement with a string as an argument

  2. valid code was ran through the executer that contained a print statement with a integer as an argument
  1. print out the string in the argument

  2. print out the integer in the argument
1The programing language SHALL have constructs that allow for a block of code to be executed or not executed based on a condition
  1. valid code was ran through the interpreter that contained an if statement with a 'true' statement as it's condition and a print statement in it's body

  2. valid code was ran through the interpreter that contained an if statement with a 'false' statement as it's condition and a print statement in it's body
  1. the code in the body executes the print statement showing it on standard out

  2. the code in the body does not execute the print statement not showing it on standard out
2The programing language SHALL be able to store primitives in temporary memory in the form of a variable
  1. A code file is ran with the code that will store a 2 in a variable and print it to the screen
  1. The code file prints out the 2 that was stored
3The programing language SHALL be able to repeat blocks of code
  1. A code file that has a print statement inside of a repeat block, with the number 5 as a parameter
  1. The print statement inside of the repeat block will run exactly 5 times
4The programing language SHALL be able to define and run functions
  1. A code file contains the function definition of a 'add' function
  1. When the defined 'add' function is ran then it will add the numbers 3 and 2 to equal 5
5The programing language SHALL have a way for functions to return values
  1. A function that adds two numbers returns a 5 to a print statement
  1. the print statement prints the 5 that was returned
6The programing language SHALL be able to check if two numbers are equal
  1. A code file contains a comparison operator returning to a print statement that is checking if 2 is equal to 2

  2. A code file contains a comparison operator returning to a print statement that is checking if 2 is equal to 3
  1. the print statement will print out 'true'

  2. the print statement will print out 'false'
7The programing language SHALL be able to check if two lists are structurally equivalent, meaning, they contain the same elements in the same order.
  1. a list which contains 1, 2 and 3 is checked against itself

  2. a list which contains 1, 2 and 3 is checked against another list containing 1, 2, 3

  3. a list which contains 1, 2 and 9 is checked against another list containing 1, 2, 3

  4. a list which contains nothing is checked against another list containing 1, 2, 3
  1. returns true

  2. returns true

  3. returns false

  4. returns false
